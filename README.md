# ReceiptMap-Japan-Docs
Receipt Map Japan 是一款專為赴日旅客設計的旅遊記帳與足跡管理工具。透過AI 辨識技術，將日文收據轉化為直觀的地圖足跡，讓您的日本之旅是一份精彩的回憶地圖。  核心功能：  AI 收據辨識：拍下日本商店收據，AI 自動提取店名、金額與消費類型，並即時轉換為中文資訊。  動態足跡地圖：自動根據收據資訊在地圖上標註消費地點，輕鬆回顧在日本的旅遊路徑。  隱私守護：我們不追蹤您的實時 GPS 位置，所有地圖標記均基於您提供的收據或手動輸入，確保隱私安全。  為什麼選擇 Receipt Map Japan？ 去日本旅行時，總是拿了一堆收據卻忘了是在哪間店買的嗎？這款 App 幫助您將「消費紀錄」與「地理位置」結合。無論是那霸的暖暮拉麵，還是國際通的藥妝店，通通幫您在地圖上標好標滿！
